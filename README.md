# 🌸 TOKO GANCI 🌸

Website toko online sederhana untuk menjual gantungan kunci handmade dengan fitur:

- 🛒 Keranjang belanja
- ⭐ Ulasan pembeli
- 💌 Checkout otomatis ke Gmail
- 💾 Penyimpanan data menggunakan LocalStorage

---

# ✨ Fitur

## 🛍️ Produk
Menampilkan produk gantungan kunci lengkap dengan:
- gambar produk
- nama produk
- harga produk

## 🛒 Keranjang Belanja
User bisa:
- menambahkan produk
- menghapus produk
- melihat total harga

## 💌 Checkout Gmail
Saat tombol checkout ditekan:
- Gmail otomatis terbuka
- isi pesanan otomatis dibuat

## ⭐ Review Pembeli
User bisa:
- mengirim ulasan
- melihat ulasan dari pembeli lain

---

# 💻 Teknologi

Project ini dibuat menggunakan:

- HTML
- CSS
- JavaScript

Tanpa framework tambahan.

---

# 🚀 Cara Menjalankan

1. Download semua file
2. Pastikan gambar produk ada di folder yang sama
3. Buka `index.html` di browser
